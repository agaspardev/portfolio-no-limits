# NO LIMITS // Cloud Operations Portfolio

Portfolio profesional de Antonio Gaspar Rengifo, construido con Next.js, React, TypeScript y Tailwind CSS.

## Enfoque

Landing page personal orientada a:

- Desarrollo web
- Integración de sistemas
- Cloud Operations
- DevOps y automatización
- Seguridad
- IA aplicada

## Estructura principal

1. Command Center
2. Profile
3. Current Focus
4. Capabilities
5. Certifications
6. Experience
7. Professional CV
8. Contact

## Datos

La información visible del sitio se mantiene separada en archivos JSON y se localiza en español e inglés.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Desarrollo

```bash
npm run dev
```
